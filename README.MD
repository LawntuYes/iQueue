# iQueue 📋

**A powerful web application for managing queues efficiently.**
### Note that this project is not deployed yet, and it was built for a college assignment.

### ⚠️ Warning: This project is not secure and should not be used in production. And this project used help from AI so there might be some bugs.
---

## 🚀 Overview

iQueue is a robust queue management system designed to streamline the process of handling queues. It provides a seamless experience for both administrators and users.

## 🛠️ Architecture

The application is split into a modern frontend and a scalable backend:

- **Frontend**: Built by **Tohar Naor** 🎨

  - Located in the `IQueue` directory.
  - Delivers a responsive and intuitive user interface.

- **Backend**: Built by **Michael Goldin** ⚙️
  - Located in the `Backend` directory.
  - Powered by **MongoDB** for reliable data storage and management.

## ✨ Features

- **Real-time Queue Management**: Efficiently track and manage queue positions.
- **MongoDB Integration**: Secure and scalable data persistence.
- **User-Friendly Interface**: Designed for ease of use.
